EcoTrack

EcoTrack is a Carbon Footprint Calculator designed to help users understand how their daily activities affect the environment.

This project supports Sustainable Development Goal 13: Climate Action by encouraging people to reduce their carbon emissions and adopt eco-friendly habits.

Why It Matters

Climate change is one of the biggest global challenges today. Small changes in lifestyle can help reduce environmental damage.

Website Features

Carbon Score System

Users receive a carbon footprint level: Low, Moderate, or High.

Personalized Eco Tips

The website gives suggestions on how users can reduce their emissions.

Save Last Result

The website automatically saves the user's last carbon footprint result using local storage.

About the Project

EcoTrack was developed as a student project using HTML, CSS, and JavaScript.

The goal of this project is to raise awareness about environmental sustainability and help users understand the impact of their lifestyle choices.
